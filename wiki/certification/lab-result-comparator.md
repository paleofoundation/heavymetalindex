Use this page when a brand, researcher, or reviewer wants to compare finished-product heavy-metal results with the structured percentile context currently available in the Heavy Metal Index.

## How To Read The Output

The comparator uses the same evidence layer as the product-category pages. For lower-is-better metals, the clean p90 column is the current candidate value at which roughly 90 percent of the selected clean comparator pool would pass. The dirty p10 column is the lower edge of the paired higher-contamination comparator when that relationship is available.

This page does not decide certification. It is a traceable pre-screen that helps identify whether a submitted lab result appears directionally consistent with the clean comparator, overlaps a known higher-contamination comparator, or needs standards review because the current public evidence is incomplete.

## What Still Requires Review

Certification review still needs the full required analyte panel, lot and sampling documentation, the original laboratory report, method and accreditation review, LOD/LOQ handling, matrix-basis matching, jurisdiction and row-fit review, and any HMT&C program rules that sit outside the public Index.
